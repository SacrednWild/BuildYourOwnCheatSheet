HTML Form Reference

Form Tags

Tag Name Description
<form> Form The element is used to collect and send information to an external source.
<input> Input The element is used to render a variety of input fields on a webpage.
<label> Label The element is used to identify a specific <input> element with label.
<select> Select The element is used to create an empty dropdown list.
<option> Option The element is used to create a dropdonw list's choice.
<datalist> Data List The element is used to implement a basic search/autocomplete functionality for a <input> element (value of id must match with <input>'s list attribute).
<textarea> Text Area The element is used to create a text-box for multi-line input.
